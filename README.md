# StreamFlix - Modern IPTV Streaming Platform

![StreamFlix Banner](https://my-streamflix.vercel.app/og-image.png)

## 📺 Overview

StreamFlix is a modern IPTV streaming platform with a Netflix-style interface. It allows users to load M3U playlists and stream live TV channels with a premium user experience.

## ✨ Features

- 🎨 **Netflix-style UI** - Premium dark interface with smooth animations
- 📱 **Responsive Design** - Works on mobile, tablet, and desktop
- 🔐 **Firebase Authentication** - Sign in with Google or Email/Password
- 💾 **Cloud Sync** - Favorites and watch history saved in the cloud
- 🎬 **HLS Streaming** - Full support for .m3u8 streams
- 📺 **M3U Playlist Support** - Load any M3U playlist URL
- ⭐ **Favorites System** - Save your favorite channels
- 📜 **Watch History** - Continue watching where you left off
- 🔍 **Search & Filter** - Find channels by name or category
- 📱 **PWA Support** - Install as native app on any device
- 🚀 **Fast Performance** - Built with Vite and React

## 🛠️ Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Bootstrap 5 + Custom CSS
- **Video Player**: HLS.js
- **Authentication**: Firebase Auth
- **Database**: Firebase Firestore
- **Animations**: Framer Motion
- **Icons**: React Icons
- **PWA**: Vite PWA Plugin
